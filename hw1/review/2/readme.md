## 我的播放器

![image](https://user-images.githubusercontent.com/58585134/110155181-3baa4580-7e20-11eb-9507-183d96b49891.png)

我的功能有以下:
1. 時間軸
    - 總時長100秒
    - 左邊的數字顯示現在秒數
    - 進度條
3. 愛心
    - 靠近愛心會放大
    - 按下去的時候會變成實心
3. 按紐
    - 滑鼠經過會變圖片
4. 上一首
    - 重置時間軸
6. 播放鍵
    - 按下播放會變成暫停 / 按下暫停會變成播放
    - 暫停時上方時間軸&進度條會跟著停止
7. 音量鍵
    - 滑鼠經過的時候會冒出選單，選單裡面有你好
